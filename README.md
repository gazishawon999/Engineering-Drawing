# 📐 Engineering Drawing Repository

This repository contains course materials, lab work, and group presentation resources for the **Engineering Drawing** course.  
It serves as a collection of practical learning outcomes, including manual sketches and AutoCAD-based exercises.

## 🧠 Learning Outcomes

During this course, the following concepts and skills were developed:

- ✏️ Manual sketching techniques in engineering drawing  
- 📏 Understanding **Orthographic Projections**  
- 🔷 Understanding **Isometric Views**  
- 💻 Practical experience with **AutoCAD software**  
- 🛠️ Familiarity with essential AutoCAD tools and commands
